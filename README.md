# random-dml

A Node.js script that randomly executes DML statement.

## Quick Run

-   Perform a random DML:

    ```
    npm start
    ```

-   Perform a batch insert:

    ```
    npm run -o insert
    ```

-   Perform a batch update:

    ```
    npm run -o update
    ```

-   Perform a batch delete:
    ```
    npm run -o delete
    ```

## Testing

To start unit testing:

```
npm test
```
